The file system

DufAdminBundle provides a fully functionnal file system. It allows you to :

  • Easily upload files in your CRUD forms
  • Manage uploaded files
  • Edit images
Configuration

If you want to use the file system, you have to configure the upload path for your assets : 

	
duf_admin:
    upload_dir: "uploads/"

Then, you have to set which extensions the users can upload : 

	
duf_admin:
    allowed_upload_extensions:
        images:
            - jpg
            - png
        documents:
            - docx
            - pdf
        videos:
            - avi
            - mp4

You can specify the number of items you want to display on each file system's page (for instance : http://your-domain/site-admin/files/images) 

	
duf_admin:
    file_system_items_per_page: 4

In your forms, the "File" and "Multiple Files" field types open modals, in which you can select the previously uploaded files, so that your users don't have to upload the same file twice.
