CRUD fields types

DufAdminBundle offers a wide variety of input types :

  • Text
  • Translatable Text
  • Date
  • Datetime
  • Email
  • Textarea (using CKEditor)
  • Translatable Textarea (using CKEditor)
  • Checkbox
  • Password
  • Choice
  • Entity
  • File
  • Multiple Files
  • Number
  • Hidden

Global parameters :

  • is_editable : boolean | required
  • type : string | required
    The type of input
  • order : integer | required
    Display order of the field relative to the form
  • label : string | optional
    The label to display above your text input
  • required : boolean | optional
    If this field must not be empty on validation, set this parameter to true

Text

Property type : string

Definition :

		
/**
 * @var string
 *
 * @ORM\Column(name="title", type="string", length=255)
 * @Editable(is_editable=true, label="Title", required=true, type="text", order=1, placeholder="Write your title")
 */
private $title;

Parameters :

  • placeholder : string | optional
    Text input placeholder

Entity

This type of input allows you to select one or more entities B, linked to your entity A with a ManyToOne or a ManyToMany relation.

select inputs are rendered using Select2, for more informations on this plugin please visit https://select2.github.io/

Property type : ManyToOne

Definition :

		
/**
 * @ORM\ManyToOne(targetEntity="AppBundle\Entity\EntityB")
 * @ORM\JoinColumn(nullable=false)
 * @Editable(is_editable=true, label="Other Entity", required=true, type="entity", order=1, relation_index="entity_b_name")
 */
 protected $entity_b;

Parameters :

  • relation_index : string | required
    name of the property of the inverse side of the ManyToOne relation. This property must be a string, as it is used in the select input as the option's name

Property type : ManyToMany

Definition :

		
/**
 * @ORM\ManyToMany(targetEntity="AppBundle\Entity\EntityB", inversedBy="entity_bs")
 * @Editable(is_editable=true, label="Other entities", required=true, multiple=true, type="entity", order=1, relation_index="title")
 */
private $entity_bs;

Parameters :

  • relation_index : string | required
    name of the property of the inverse side of the ManyToMany relation. This property must be a string, as it is used in the select input as the option's name
  • multiple : string | required
    allows you to select multiple entities in the select input

File

file inputs are displayed in a modal. The file input itself if rendered using Bootstrap File Input. For more documentation on this plugin, please visit http://plugins.krajee.com/file-input.

Property type : ManyToOne

Definition :

		
/**
 * @ORM\ManyToOne(targetEntity="Duf\AdminBundle\Entity\File")
 * @ORM\JoinColumn(nullable=true)
 * @Editable(is_editable=true, label="Cover", required=false, type="file", filetype="images", order=1)
 */
 protected $cover;

Parameters :

  • filetype : string | required
    Type of file to be uploaded : images, videos or documents
