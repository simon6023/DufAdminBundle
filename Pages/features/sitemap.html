Sitemap

DufAdminBundle offers you a simple way to generate your sitemap.xml file, using custom annotations in your entities. The sitemap file will be automatically available at http://yourdomain/sitemap.xml

The SitemapableAnnotation takes several parameters :

  • route : the name of the route used to create the urls in the sitemap. For example, if you have an Article entity, you can decide to generate URLs for every article in the sitemap.
  • route_params : if your route takes parameters, indicate it here in an array : keys are the name of the route parameter, values are the name of the entity property.
  • find_by : if you want to filters the entities displayed in the sitemap, you can indicate it here.
  • changefreq : the URL changefreq
  • priority : the URL priority

To use this feature, update your entity class : 

	
// AppBundle/Entity/YourEntity

use Duf\CoreBundle\Annotations\SitemapableAnnotation as Sitemapable;

/**
 * YourEntity
 *
 * @ORM\Table(name="your_entity")
 * @ORM\Entity()
 * @Sitemapable(route="your_route_name", route_params={"slug":"slug"}, find_by={"enabled":"1"}, changefreq="daily", priority="1.0")
 */
class YourEntity extends DufAdminEntity implements Translatable
{
    /**
     * @var string
     *
     * @ORM\Column(name="title", type="string", length=255)
     */
    private $title;

    /**
     * @ORM\Column(length=128, unique=true)
     */
    private $slug;
}

If you want to display static URLS in your sitemap, simply add the routes in your config.yml file : 

	
# app/config/config.yml

duf_core:
    static_urls:
        - who_is
