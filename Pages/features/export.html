Export

Basic usage

Using the default CRUD index page, an export feature is available at the bottom of the list table. Two formats are available right now, CSV and Excel (.xls).

To enable this feature for your entity, you have to configure your entity. First, import the annotation : 

	
// AppBundle\Entity\YourEntity.php

use Duf\AdminBundle\Annotations\ExportableAnnotation as Exportable;

Then, on each property you want to export, add the annotation like this : 

	
/**
 * @var string
 *
 * @ORM\Column(name="title", type="string", length=255)
 * @Exportable(is_exportable=true, label="Title", type="text")
 */
private $title;
Export field types

For each property type (text, boolean, entity relationship...), an export type is defined :

Global parameters :

  • is_exportable : boolean | required
  • type : string | required
    The type of input
  • label : string | required
    The label to display as your column header

Text

Property type : text

Definition :

		
/**
