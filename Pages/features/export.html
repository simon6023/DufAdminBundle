Export

Basic usage

Using the default CRUD index page, an export feature is available at the bottom of the list table. Two formats are available right now, CSV and Excel (.xls).

To enable this feature for your entity, you have to configure your entity. First, import the annotation : 

	
// AppBundle\Entity\YourEntity.php

use Duf\AdminBundle\Annotations\ExportableAnnotation as Exportable;

Then, on each property you want to export, add the annotation like this : 

	
/**
 * @var string
 *
 * @ORM\Column(name="title", type="string", length=255)
 * @Exportable(is_exportable=true, label="Title", type="text")
 */
private $title;
Export field types

For each property type (text, boolean, entity relationship...), an export type is defined :

Global parameters :

  • is_exportable : boolean | required
  • type : string | required
    The type of input
  • label : string | required
    The label to display as your column header

Text

Property type : text

Definition :

		
/**
 * @var string
 *
 * @ORM\Column(name="title", type="string", length=255)
 * @Exportable(is_exportable=true, label="Title", type="text")
 */
private $title;
Boolean

Property type : boolean

Definition :

		
/**
 * @var bool
 *
 * @ORM\Column(name="enabled", type="boolean")
 * @Exportable(is_exportable=true, label="Enabled", type="boolean")
 */
private $enabled;
Entities Collection

Use this type to export the content of a collection property. For example, a blog article with multiple tags. All the tags will be displayed in one single column in the export file.

Property type : boolean

Definition :

		
/**
 * @ORM\ManyToMany(targetEntity="AppBundle\Entity\YourOtherEntity", inversedBy="your_entity")
 * @Exportable(is_exportable=true, label="Your Label", type="entities_collection", relation_field="title")
 */
private $otherEntities;

Parameters :

  • relation_field : string | required
    The name of the property of the targetEntity. The value of this property will be displayed in the exported file

Relation

Property type : ManyToMany | ManyToOne | OneToOne

Definition :

		
/**
 * @ORM\ManyToOne(targetEntity="AppBundle\Entity\YourOtherEntity")
 * @ORM\JoinColumn(nullable=true)
 * @Exportable(is_exportable=true, label="Your Label", type="relation", relation_field="title")
 */
 protected $entity;

Parameters :

  • relation_field : string | required
    The name of the property of the targetEntity. The value of this property will be displayed in the exported file

Count

Property type : array | ManyToMany

Definition :

		
/**
 * @var array
 *
 * @ORM\Column(name="collection", type="json_array")
 * @Exportable(is_exportable=true, label="Collection", type="count")
 */
private $collection;
