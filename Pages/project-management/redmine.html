Redmine

DufAdminBundle provides an implementation of the Redmine API, using kbsali/redmine-api Bundle. For more information on this bundle, please visit the github repo.

Configuration

To use this feature, simply define some configuration variables : 

	
# AppBundle/config/config.yml

duf_admin:
    redmine:
        host: http://your-redmine-host/
        project: your-project
        api_key: your-api-key

Then you must set the Redmine username to each user. To do so, simply edit the right user (select "Users" in the navigation menu), and edit the "Redmine Username" field.

When the user authenticates to the admin section of your website, the Redmine link will be available in the navigation menu :

Usage

On the list view (http://your-domain/site-admin/redmine), you'll find every issue existing for the project set in the configuration.

For the moment, you can only create new issues, but you can't edit an issue. This will be fixed soon.
