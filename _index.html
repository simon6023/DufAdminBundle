DufAdminBundle

Symfony3 Admin Bundle

View on GitHub Download .zip Download .tar.gz

Description

The purpose of this bundle is to provide a fully functional admin application. This bundle is a ready-to-use solution to integrate quickly and easily customizable CRUDs, by using entity annotations and yml configuration files.

Out of the box, the bundle also comes with some cool features : translations support, file uploads and file management, image editing, messaging system, Redmine integration, sitemap generator, notification listeners...
